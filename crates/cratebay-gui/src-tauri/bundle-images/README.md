# Bundle Images

Pre-built Docker images for CrateBay sandboxes.

These files are generated by `scripts/build-bundle-images.sh` and bundled into the application.
They are not checked into git — run the build script to generate them.

## Generate

```bash
./scripts/build-bundle-images.sh
```

## Images

| File | Image | Base |
|------|-------|------|
| python-dev.tar.gz | cratebay-python-dev:v1 | python:3.12-slim-bookworm |
| node-dev.tar.gz | cratebay-node-dev:v1 | node:20-slim |
| rust-dev.tar.gz | cratebay-rust-dev:v1 | rust:1-slim-bookworm |
| ubuntu-base.tar.gz | cratebay-ubuntu-base:v1 | ubuntu:24.04 |
